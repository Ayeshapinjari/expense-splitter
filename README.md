
# Expense Splitter

A simple Expense Splitter web application built using HTML, CSS, and JavaScript.

## Features

* Create groups
* Add members
* Add expenses
* Split expenses equally
* Shows who owes whom
* Saves data using LocalStorage
* Responsive design

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the project
2. Open the project folder
3. Open `index.html` in browser
